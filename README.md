# Replication Package
This is the replication package for the thesis "The Role of Context in Automated Requirements-to-Requirements Traceability Link Recovery" created by David Bauch at the Karlsruhe Institute of Technology.

As part of this replication package, both the [Code](./Code/) created as part of the thesis, as well as the obtained evaluation results and configurations ([see here](./Evaluation/)) are published.
For more information, check the README file within the specific directories.
